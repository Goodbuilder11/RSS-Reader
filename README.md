# RSS Reader

## Description
This project is an RSS feed reader that fetches and displays the title, link, and description of an RSS feed from given website URLs.


Formatted with Pylint




## Usage
Uses 'uv' package manager

uv run main.py


## Dependencies

- Python 3.10+
- feedparser 6.0.11
- pylint 3.3.5
- ruff 0.9.10
